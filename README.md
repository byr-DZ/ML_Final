# ML_Final
大家好，这是我机器学习课程的期末大作业！
在该工程文件包括四个部分，分别为：

## fashionmnist
这个文件夹里面为此次实验的数据集，FashionMNIST数据集，在项目中可以直接加载。
## images
该文件夹中保存的是本次实验中产生的所有图片。
## Analysis on FashionMnist.ipynb
这是本次大作业的第一部分，研究了不同训练参数和策略对同一模型方法的性能影响。
## DModel Performance on FashionMNITST.ipynb
这是本次大作业的第二部分，研究了在同一数据集上的不同模型的性能表现。

上述所有文件可以直接在jupyter中运行，运行环境为Python3.5和与之匹配的Tensorflow环境。
